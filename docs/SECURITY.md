# DevilChain Security Policy

## Reporting Vulnerabilities

Do NOT open a public issue for security vulnerabilities.
Email: devilchain@devilpvt.in
Or open a private GitHub Security Advisory.

## Wallet Security
| Layer | Technology |
|---|---|
| Encryption | AES-256-GCM |
| Signing | Ed25519 |
| Key Exchange | Curve25519 |
| Key Derivation | BIP39 + BIP32 HD |

## Network Security
- DDoS Protection via rate limiting + AI anomaly detection
- Anti-Sybil: Stake-based identity + DAO approval
- Validator Reputation: On-chain scoring
- Smart Contract Audit: DevilGuard AI on all deployments

## DevilGuard AI
- Rug pull pattern detection
- Scam token Graph AI analysis
- Bot/spam NLP filtering
- Transaction anomaly detection (ONNX model)

## Bug Bounty
DevilChain DAO may award DVC bounties for critical disclosures.
